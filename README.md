## git 、 git hub

### git init

### git status

查看目前資料夾狀態
```git
git add status
```

### git add

全部加入暫存區
```git
git add .
```

指定加入暫存區
```git
git add ./{要 add 的檔案路徑}
```

### git config

設定全域使用者名稱、信箱。
```git
git config --global user.name "你的名字"
git config --global user.email "你的Email"
```

查看目前設定。
```git
git config --list
```

### git commit

提交
```git
git commit -m "{提交訊息}"
```
